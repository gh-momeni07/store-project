# معرفی پروژه

این پروژه با زبان برنامه‌نویسی Python و با استفاده از برنامه‌نویسی شی‌گرا (OOP) پیاده‌سازی شده است.

هدف پروژه، شبیه‌سازی یک سیستم مدیریت فروشگاه اکسسوری است که دو نوع کاربر (مدیر و مشتری) دارد و امکان مدیریت محصولات و سفارش‌ها را فراهم می‌کند.

اطلاعات محصولات، کاربران و سفارش‌ها در فایل‌های CSV ذخیره می‌شوند.

# کنترل خطا

در پروژه کنترل خطاهای زیر انجام شده است:

- جلوگیری از ورود قیمت غیرعددی
- جلوگیری از ورود موجودی غیرعددی
- جلوگیری از وارد کردن موجودی منفی
- بررسی وجود محصول
- بررسی کافی بودن موجودی کالا
- بررسی صحت نام کاربری و رمز عبور
- جلوگیری از ثبت نام با نام کاربری تکراری

# نحوه اجرای پروژه

1. Python 3 روی سیستم نصب باشد.
2. تمام فایل‌های پروژه در یک پوشه قرار داشته باشند.
3. فایل `final_project.py` اجرا شود.
4. برای نسخه نهایی دارای رابط گرافیکی، فایل مربوط به Tkinter اجرا شود.
